Getting Started

What is My Profile?

  • My Profile is an application that enables you to easily view and update your Profile settings.
  • A Profile allows you to control which transactions you receive email notifications of, which are visible to you, and which you can act on.
  • All users who have access to Web Suite applications are required to maintain their Profile to view and act on transactions.
  • New approved Web Suite users will have their Profile setting auto map to all Intel Locations. To set any preference, users would need to modify My Profile settings by deselecting the Intel locations.

    Note: If there is any change for your company name or address, kindly contact Intel buyer to make necessary change in Supplier’s database as well.

How does My Profile work?

  • Go to nsupplier.intel.com.
  • Supplier update his/her personal and company details.
  • Selected/filtered PO will be visible to supplier.

What are the key features?

  • View Profile tab
    • To view your as-is Profile settings.
    • Edit Profile tab To perform search then update your Profile settings, through a specific PO number.
  • Edit Profile tab
    • Supplier location available on your account.
    • OR, defined Intel location.
  • Company Information tab
    • To update your company information.

 

View Profile

How do I view my current profile settings?

  • Go to nsupplier.intel.com. 
  • Click on Supplier Sign-In.
    Result: The Intel Login page will be displayed.
  • Enter your User ID and Password after click Continue to Sign In or Register if you don’t have an account.
  • Click Sign In.
    Result: The Supplier Information page will be displayed.
  • Click Intel Web PO/Web Invoice/Payment Tracker > My Profile link.
    Result: The View Profile tab will be displayed.
    Note : Click Edit Profile button to make changes to your current settings.

What should I do with “incomplete Company Information” error message?

Incomplete Company Information error could be a result of incomplete TAX information, company address, or city. 

  • Go to My Profile > Company Information tab.
  • Update Local Company Address.
  • Update Corporate Address.
    Notes:
    • Corporate Address is the address Intel has on file for your company.
    • It is necessary to use the address Intel has for you in order to properly process your application.
    • Contact your Intel buyer for further confirmation.
  • Update TAX/VAT/Company Reg. Number.
  • Click Save Profile button.
    Results:
    • Your company information will be updated accordingly.
    • You should be able to view your profile in My Profile > View Profile tab.

What should I do with “incorrect profile setting” error message?

Incorrect profile setting could be a result of mismatched Vendor ID-Company Code combination.

 

What should I do with “company’s location(s)” error message?

  • Go to My Profile.
  • Click Edit Profile tab.
    Result: The Edit Profile page will be displayed.
    Note: You can display the list of vendor and company code base on a specific PO, vendor name, or company code.
  • Select option from dropdown.
    Notes: Step-by-step instructions will be displayed on screen.
  • Select supplier location-to-Intel location combination.
  • Click Submit Change button.
    Results:
    • You have successfully completed updating your user profile.
    • You should be able to view your profile in My Profile > View Profile tab.

Editing Profiles

How do I update My Profile settings?

  • Go to My Profile.
  • Click Edit Profile tab.
    Result: The Edit Profile page will be displayed.
    Note: You can display the list of vendor and company code base on a specific PO, vendor name, or company code.
  • Select option from dropdown.
  • Update the required information.
    Note: Select supplier location-to-Intel location combination.
  • Click Submit Change button.
    Results:
    • You have successfully completed updating your user profile.
    • The View Profile tab will be displayed.

How do I filter/select my viewable transactions?

Supplier Locations

  • Called Vendor or Supplier Number.
  • 10-digits number.
  • For example, 0123450001, 0123450002, etc.

Intel Locations

  • Called Company Code.
  • 3-digits number.
  • For example, 100 – Intel Corporation, 150 – Intel Americas Inc, etc.

 

How does the filtering combinations work?

Your Profile allows you to filter out transactions you do not wish to see. This means you don’t have to look through lists of transactions to find the ones that you’re responsible for. This also enables you to share the transaction workload with other users from your company.

Note: It is most efficient if all users at a company can jointly agree on Profile settings to ensure proper ‘ownership’ of all transactions. Failure to manage Profiles across all users can leave some transactions un-owned and can impact the supply line.

 

How do I update my profile using a specific PO number?

  • Go to My Profile.
  • Click Edit Profile tab.
    Result: The Edit Profile page will be displayed.
    Note: You can display the list of vendor and company code base on a specific PO, vendor name, or company code.
  • Select PO option.
  • Enter PO number.
    For example: PO number 4300123456 generated from Intel location 150 to supplier location 0123450001.
    Notes:
    • This indicates you would like to view PO’s similar to this PO number 4300123456.
    • Or in other words, from the combination of Intel location 150 to supplier location 0123450001.
  • Click show vendors button.
  • Select supplier location to Intel location combination.
    Note: This indicates you would like to view PO’s placed from Intel location 150 to supplier location 0123450001.
  • Click Submit Change button.
    Results:
    • You have successfully updated your user profile to view all PO’s from Intel location 150 to supplier location 0123450001.
    • The View Profile tab will be displayed.

How do I update my profile using supplier location?

  • Go to My Profile
  • Click Edit Profile tab.
    Result: The Edit Profile page will be displayed.
    Note: You can display the list of vendor and company code base on a specific PO, vendor name, or company code.
  • Select Vendor option.
  • Click combo box.
    Result : The list of supplier locations available on your account will be displayed accordingly.
  • Select supplier location. 
    For example: 0123450001.
    Note: This indicates you would like to view PO’s placed to this supplier location.
  • Click show vendors button.
  • Select supplier location to Intel location combination. 
    For example: Intel Americas, Inc. – 150.
    Note : This indicates you would like to view PO’s placed from Intel location 150 to supplier location 0123450001.
  • Click Submit button.
    Results:
    • You have successfully completed updating your user profile to view all PO’s from Intel location 150 to supplier location 0123450001.
    • The View Profile tab will be displayed.

How do I update my profile using Web PO?

  • Go to Intel Web PO.
  • Enter related keywords to specify and narrow down the search criteria.
    Result: The search result with matching criteria will be generated.
    Note: You need to modify your profile setting in order to view more search results.
  • Click Add PO to Profile.
    OR,
  • Click to view complete list link to view and select PO from a pop up window.
    Results:
    • If successful, the acknowledgement message will be displayed.
    • Error message will be displayed.
  • Note: Should you received the error message, kindly validate your profile D-U-N-S number before contacting Intel Technical Support.
    • It takes 1 to 2 minutes to update your profile to the system.
    • Error message will be generated in case the same Supplier
    • Number/Vendor ID has already clicked on the same PO and still pending for approval.

How do I update my email address and other personal data?

  • Go to nsupplier.intel.com. 
  • Click on Supplier Sign-In.
    Result: The Intel Login page will be displayed.
  • Enter your User ID and Password after click Continue to Sign In or Register if you don’t have an account.
  • Click Manage My Account link.
  • Click Sign In.
    Result: The Manage My Account page will be displayed.
  • Click Account Information link.
    Result: The Account Information page will be displayed.
  • Update your personal information.
  • Click Submit button.
    Result: Your personal information has been updated successfully.
